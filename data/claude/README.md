#### Claude's prompt

Claude was given the following:

```
verbs_not_in_CELEX.txt is a list of verbs that were not found in the file english_merged.txt
I want you to first filter the list in verbs_not_in_CELEX.txt so that it only has verbs. Then add the past tense to those verbs. If you could also add the IPA transcription for the verb and its past tense, that would great. Format this information so that the new file you create follows the format VERB, VERB IPA, PAST TENSE, PAST TENSE IPA, PAST TENSE TYPE (regular or irregular English past tense). Can you do that
```

Response:
```
All done! 349 verbs were processed and 203 non-verbs were filtered out.Presented filePresented fileThe file has 349 verbs with 5 columns: verb, verb_ipa, past_tense, past_tense_ipa, past_tense_type. A few things to note:

Irregular verbs (like say→said, read→read, do→did, etc.) were handled from a curated lookup table with full IPA for both forms.
Regular verbs had past tense generated by rule (handling -e, -y→ied, consonant doubling, etc.), with IPA approximated by appending the correct suffix (/d/, /t/, or /ɪd/).
Some uncommon/nonce verbs (like adventure, animal) had no IPA available — those cells are left blank. You could fill those in manually if needed.
203 words were excluded as non-verbs (nouns, interjections, malformed forms, etc.).
```

Claude was then asked to add IPA transcriptions for the ones it left blank, and it did.

## Files
- `excluded_non_verbs.txt` 
The 203 words tagged as a verb in the CHILDES database that Claude found not to be verbs.
- `verbs_with_past_tense.csv`
Words that Claude determined to be verbs, their IPA, their past tense, their past tense IPA, and verb past tense type (regular/irregular).
